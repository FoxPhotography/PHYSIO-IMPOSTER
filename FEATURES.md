# 🎮 Feature Overview

## Complete Game Flow

### 1. 🎭 Install Prompt (First Visit)
**What you see:**
- Awesome animated install modal
- App icon with pulsing animation
- "PHYSIO IMPOSTER" title in red
- Three feature highlights with emojis
- "INSTALL NOW" button (red gradient)
- "Maybe Later" option

**Features:**
- Only shows on first visit
- Can be dismissed
- Triggers native install prompt
- Works on Chrome/Edge/Safari

---

### 2. 💪 Category Selection
**What you see:**
- Large "PHYSIO IMPOSTER" title with glow effect
- Three category cards:
  - 💪 Upper Limb (Shoulder, Arm & Hand)
  - 🦵 Lower Limb (Hip, Leg & Foot)
  - 🔥 All Muscles (Complete Challenge) - highlighted in red

**Features:**
- Smooth fade-in animations
- Hover effects with red glow
- Cards slide up on tap
- Modern gradient backgrounds

---

### 3. 👥 Player Setup
**What you see:**
- Back button (top left)
- "Player Setup" header
- Player counter with +/- buttons
  - Large red circular buttons
  - Number display in center (3-12)
  - Disabled state when at limits
- Input fields for each player
  - Auto-generated placeholders (Player 1, Player 2...)
  - Smooth scrolling for many players
  - Red focus glow effect
- "START GAME" button (pulsing)

**Features:**
- Min 3, max 12 players
- Auto-fills names if left empty
- Smooth animations
- Mobile keyboard friendly
- Custom scrollbar (red theme)

---

### 4. 🎯 Game Introduction
**What you see:**
- 🎭 Large mask icon
- "Ready to Play?" title
- Instructions text
- "One of you is the IMPOSTER!" warning (red)
- Player list with numbers
- "BEGIN GAME" button (pulsing)

**Features:**
- Shows all players
- Clear instructions
- Builds anticipation
- Smooth transitions

---

### 5. 🔄 Role Reveal (Pass & Play)
**What you see:**

**Before Reveal:**
- Progress bar (shows X/Total)
- 👤 Player icon
- Player name in large text
- "It's your turn!" message
- "Tap to see your role" instruction
- "REVEAL ROLE" button (pulsing red)

**After Reveal (Crewmate):**
- 💪 Muscle icon
- "YOUR MUSCLE IS" label
- Muscle name (green, large)
- "Find the imposter among you!" hint
- Green glowing border
- "NEXT PLAYER" button

**After Reveal (Imposter):**
- 🎭 Mask icon
- "YOU ARE THE" label
- "IMPOSTER" text (red, glowing)
- "Blend in and don't get caught!" hint
- Red pulsing border
- "NEXT PLAYER" button

**Features:**
- One player at a time
- Privacy-focused (hide before passing)
- Clear visual distinction
- Smooth animations
- Progress tracking

---

### 6. 🎊 Game End
**What you see:**
- Last player's role card
- "All players have seen their roles!" message
- "Now discuss and find the imposter!" instruction
- Two buttons:
  - "PLAY AGAIN" (red, same players)
  - "NEW GAME" (outlined, back to setup)

**Features:**
- Quick replay option
- Easy restart
- Clear next steps
